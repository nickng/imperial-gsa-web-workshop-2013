Structure of an HTML page

  • Start with a 'doctype'
  • Define a header
  • Define the body
    • Divide into sections with headers (<h1>..<h6>)
    • Blocks (<div>)
    • Inline elements (<span>)
<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
    <h1>Header 1<h1>
    <h2>Header 2<h1>
    <h1>Header 3<h1>
    <h2>Header 4<h1>
    <h1>Header 5<h1>
    <h2>Header 6<h1>
  <body>
  </body>
</html>

HTML styling

<strong>Strong text</strong> (or <b>bold</b>)
<em>Emphasis</em> (or <i>italics</i>)
<u>Underline</u>
